# Oragir Parser

Автоматический парсер школьного электронного дневника **e-diary.emis.am** с уведомлениями в Telegram.

---

## Что делает

- Заходит в электронный дневник ребёнка
- Проверяет оценки за все месяцы текущего полугодия (с января по текущий месяц)
- Сравнивает с ранее сохранёнными оценками
- Отправляет **только новые** оценки в Telegram
- Запускается автоматически 3 раза в день (10:00, 15:00, 20:00)

---

## Требования

- Node.js 18+
- Google Chrome (устанавливается автоматически через Puppeteer)

---

## Установка

```bash
git clone <repo-url>
cd Oragir_parser
npm install
```

---

## Настройка

Создай файл `.env` в корне проекта:

```env
EMAIL=your_email@gmail.com
PASSWORD=your_password
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
CHILD_NAME=ИМЯ_РЕБЁНКА
```

### Как получить Telegram данные

1. Открой в Telegram: `@BotFather`
2. Напиши `/newbot`, дай имя и username
3. Скопируй **токен бота** в `TELEGRAM_BOT_TOKEN`
4. Напиши боту любое сообщение (чтобы активировать чат)
5. Напиши `@userinfobot` чтобы узнать свой **chat ID** — запиши в `TELEGRAM_CHAT_ID`

## Запуск

### Одноразовая проверка

```bash
npm run check
```

### Автоматический режим (3 раза в день)

```bash
npm start
```

Расписание по умолчанию: **10:00, 15:00, 20:00**.

---

## Первый запуск

При первом запуске откроется окно браузера со страницей логина. Скрипт автоматически заполнит email и пароль. Если reCAPTCHA покажет задачу с картинками — **реши её вручную** в открытом окне браузера.

После успешного входа сессия сохраняется в папке `chrome_profile/`. Последующие запуски не требуют повторного логина (пока сессия активна).

## Формат уведомлений

При обнаружении новых оценок бот отправляет сообщение:

```
Новые оценки:

05.02.2026 | Մաթեմատիկա | Оценка: 9
06.02.2026 | Անգլերեն | Оценка: 8
```

---

## Заметки

- Парсер игнорирует нечисловые оценки (например "բ")
- Если сессия истекла, при следующем запуске потребуется снова решить reCAPTCHA вручную
- Файл `grades.json` хранит историю всех найденных оценок — не удаляй его, иначе все оценки будут отправлены заново
